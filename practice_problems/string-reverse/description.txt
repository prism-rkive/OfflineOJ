Problem Title: String Reverse

Description:
You are given a string. Your task is to write a program that reverses the string and prints it.

Input:
- A single line containing the string s (1 ≤ |s| ≤ 1000), consisting of only lowercase English letters.

Output:
- Print the reversed string.

Example:
Input:
hello

Output:
olleh

Constraints:
- The input string will not contain spaces.
