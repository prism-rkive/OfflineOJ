Problem Title: Factorial Number

Description:
Calculate the factorial of a given non-negative integer n. The factorial of n (denoted as n!) is the product of all positive integers less than or equal to n.

Input:
- A single integer n (0 ≤ n ≤ 20).

Output:
- A single integer representing n!.

Example:
Input:
5

Output:
120

Constraints:
- 0 ≤ n ≤ 20
