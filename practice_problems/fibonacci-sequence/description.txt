Problem Title: Fibonacci Sequence

Description:
Print the first n numbers of the Fibonacci sequence. The sequence starts with 0 and 1, and each subsequent number is the sum of the previous two.

Input:
- A single integer n (1 ≤ n ≤ 50).

Output:
- The first n Fibonacci numbers separated by space.

Example:
Input:
7

Output:
0 1 1 2 3 5 8

Constraints:
- 1 ≤ n ≤ 50
